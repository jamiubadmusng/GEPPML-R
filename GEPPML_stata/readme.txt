This file describes how to perform GEPPML of the CESifo Working paper No. 5592
"Estimating General Equilibrium Trade Policy Effects: GE PPML" by James E. Anderson, Mario Larch and Yoto V. Yotov available for download at 
http://www.cesifo-group.de/de/ifoHome/publications/working-papers/CESifoWP/CESifoWPdetails?wp_id=19172928.

geppml.do:
+ This file runs Steps 1 to 3, i.e., the `baseline' scenario, the `conditional' scenario as well as the `full endowment' scenario.
+ It uses as data input the file ge_ppml_data_orig.dta.
+ Additionally, ppml version 2.2.2. (from October 10th, 2015) or later has to be installed. ppml can be installed by typinng "ssc install ppml" in Stata.
  Please see for details to ppml.ado the following webpage: http://personal.lse.ac.uk/tenreyro/LGW.html.
