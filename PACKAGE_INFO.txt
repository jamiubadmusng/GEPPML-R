WHAT THIS PACKAGE DOES
================================================================================

This package provides a complete R translation of the GEPPML methodology from:

  Anderson, J. E., Larch, M., & Yotov, Y. V. (2018)
  "GEPPML: General equilibrium analysis with PPML"
  The World Economy, 41(10), 2750-2782

Key Features:
  âœ“ Estimates structural gravity models with PPML
  âœ“ Computes general equilibrium trade policy effects
  âœ“ Three scenarios: Baseline, Conditional GE, Full GE
  âœ“ Fast implementation using modern R packages (fixest, data.table)
  âœ“ Produces identical results to original Stata version
  âœ“ Publication-ready outputs

================================================================================
QUICK START (3 STEPS)
================================================================================

1. INSTALL PACKAGES
   R> install.packages(c("data.table", "fixest", "haven", "dplyr"))

2. RUN ANALYSIS
   R> setwd("path/to/GEPPML_R")
   R> source("geppml.R")

3. VIEW RESULTS
   R> results <- read.csv("output/all_indexes_geppml.csv")
   R> View(results)

That's it! You now have country-level welfare effects and GE indexes.
